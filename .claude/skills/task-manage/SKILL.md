# タスク管理スキル

## 前提
- タスク管理ファイルのパスと完了マーク形式は **CLAUDE.md の `## Task Management` セクション** を参照すること
- CLAUDE.md に定義がない場合はユーザーに確認する

## トリガーと手順

### 「タスクを追加してください」
1. CLAUDE.md から `task_file`（タスク管理ファイルパス）を取得
2. タスク管理ファイルを読み込む（存在しない場合は作成）
3. ユーザーの指示に基づき、未完了形式でタスクを追記
4. 進捗サマリーがある場合は数値を更新

### 「タスクを更新してください」/ 「タスクを完了にしてください」
1. CLAUDE.md から `task_file` と `done_marker` を取得
2. タスク管理ファイルを読み込む
3. 指定されたタスクの未完了マークを完了マークに変更
4. 進捗サマリーがある場合は数値を更新（完了数 / 総数）
5. 変更内容をユーザーに報告

### 完了マーク形式（CLAUDE.md で定義）
- チェックボックス形式: `- [ ]` → `- [x]`
- 絵文字形式: `⬜` → `✅`
- その他プロジェクト固有の形式

### 協力者情報
- タスク管理ファイル内に協力者の記録がある場合、リリース時の Acknowledgements 参照用として保持する
